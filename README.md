# Sales Analysis Project

## Overview
This project focuses on analyzing regional sales data to identify trends, evaluate product performance, and generate meaningful business insights. The objective is to demonstrate a complete data analysis workflow, including data cleaning, exploration, querying, and visualization.

The analysis combines Python, SQL, and Power BI to transform raw data into an interactive dashboard that supports data-driven decision making.

---

## Tools and Technologies
Python (Pandas, NumPy)  
SQL  
Power BI  
Microsoft Excel  
Jupyter Notebook  

---

## Project Workflow
1. Data collection from sales datasets
2. Data cleaning and preprocessing using Python
3. Exploratory data analysis using Python and SQL
4. Data visualization and dashboard creation using Power BI
5. Extraction of key business insights

---

## Power BI Dashboard
The Power BI dashboard provides interactive visualizations to analyze sales performance and profitability.

Key dashboard components include:
- Total sales overview
- Profit analysis by product
- Sales distribution by region
- Location-based sales insights
- Detailed product performance analysis

## Key Insights
- Sales performance varies significantly across regions.
- Certain products generate higher profits despite lower sales volume.
- Location-based sales patterns reveal potential opportunities for business growth.
- Product category analysis helps identify top-performing products.
